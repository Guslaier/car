# Car (ESP32 Wi-Fi RC Car)

โปรเจกต์นี้คือรถบังคับด้วย ESP32 ผ่านหน้าเว็บ โดยตัว ESP32 จะปล่อย Wi-Fi แบบ Access Point ชื่อ `Car001` แล้วเปิดหน้าเว็บควบคุมทิศทาง (เดินหน้า/ถอยหลัง/เลี้ยว/หยุด) ที่ส่งคำสั่งไปยังมอเตอร์ผ่านไดรเวอร์ H-Bridge เช่น L298N

## ความสามารถหลัก

- ควบคุมรถผ่านมือถือหรือคอมพิวเตอร์โดยไม่ต้องใช้อินเทอร์เน็ต
- หน้าเว็บควบคุมแบบปุ่มกดค้าง: ปล่อยปุ่มแล้วรถหยุด
- ทำงานทั้งเมาส์และหน้าจอสัมผัส
- ESP32 ทำงานเป็น Wi-Fi AP ในตัว


## อุปกรณ์ที่ใช้

- ESP32 Dev Module
- Motor Driver (เช่น L298N หรือ H-Bridge เทียบเท่า)
- มอเตอร์ DC 2 ตัว (ซ้าย/ขวา)
- แหล่งจ่ายไฟที่เหมาะสมกับมอเตอร์และบอร์ด
- สายไฟและโครงรถ

## การต่อขา (ตามโค้ดปัจจุบัน)

ใน `Car.ino` กำหนด GPIO ดังนี้:

- `IN1 = GPIO26` ซ้ายเดินหน้า
- `IN2 = GPIO25` ซ้ายถอยหลัง
- `IN3 = GPIO33` ขวาเดินหน้า
- `IN4 = GPIO32` ขวาถอยหลัง

เชื่อมขาเหล่านี้เข้ากับขาอินพุตของไดรเวอร์มอเตอร์ให้ตรงกัน

## การเตรียม Arduino IDE

1. ติดตั้ง Arduino IDE
2. เพิ่ม Board Manager URL ของ ESP32:

	```
	https://dl.espressif.com/dl/package_esp32_index.json
	```

3. ไปที่ Boards Manager แล้วติดตั้ง `esp32` โดย Espressif
4. เลือกบอร์ดเป็น `ESP32 Dev Module`

## การอัปโหลดโปรแกรม

1. เปิดไฟล์ `Car.ino`
2. เลือกพอร์ตของบอร์ด ESP32
3. กด Upload
4. เปิด Serial Monitor (115200 baud) เพื่อตรวจสอบสถานะ

## วิธีใช้งาน

1. เปิดเครื่อง ESP32
2. เชื่อมต่อ Wi-Fi ชื่อ `Car001`
3. เปิดเบราว์เซอร์ไปที่:

	- `http://192.168.4.1`

4. กดปุ่มบนหน้าเว็บเพื่อควบคุมรถ

### คำสั่งที่ระบบรองรับ

- `forward` เดินหน้า
- `backward` ถอยหลัง
- `left` เลี้ยวซ้าย
- `right` เลี้ยวขวา
- `stop` หยุด

Backend endpoint:

- `GET /control?dir=<direction>`
